# Baldwin Nurseries Website

A responsive, modern website for Baldwin Nurseries built with HTML, CSS, and minimal JavaScript.

## Features

- **Responsive Design**: Works perfectly on desktop, tablet, and mobile devices
- **Modern UI**: Clean, nature-inspired design with green theme
- **Image Slider**: Hero section with navigation arrows and pagination dots
- **Sticky Navigation**: Header stays visible while scrolling
- **Social Media Integration**: Facebook link with icon
- **Video Player**: Placeholder for promotional videos
- **Minimal JavaScript**: Only essential slider functionality
- **Accessibility**: Keyboard navigation and screen reader friendly

## Customization

### Colors
The main green color used throughout the site is `#228b22` (forest green). You can easily change this in the CSS file by updating the color values.

### Images
The hero slider currently uses a placeholder image from Unsplash. To add your own images:
1. Replace the `src` attribute in the hero section
2. Add multiple slides by duplicating the `.hero-slide` div
3. Update the pagination dots accordingly

### Content
- Update contact information in the HTML
- Add your actual catalogue PDF file
- Replace the video placeholder with your promotional video

### Navigation
Add new pages by:
1. Creating new HTML files
2. Updating the navigation links in the header
3. Adding corresponding CSS if needed
